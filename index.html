Responsive and lightweight grid for your sites with initstyle

For now, initstyle is a simplest grid system based on Flexbox. Coming soon will be a css framework with components and classes.

Basic example:

<div class="row">
  <div class="col sm-12 md-4 lg-3 xl-4">
    <div class="block"></div>
  </div>
  <div class="col sm-12 md-8 lg-4 xl-4">
    <div class="block"></div>
  </div>
  <div class="col sm-12 md-12 lg-5 xl-4">
    <div class="block"></div>
  </div>
  <div class="col sm-12 md-7 lg-4 xl-2">
    <div class="block"></div>
  </div>
  <div class="col sm-12 md-5 lg-5 xl-5">
    <div class="block"></div>
  </div>
  <div class="col sm-12 md-12 lg-3 xl-5">
    <div class="block"></div>
  </div>
</div>
To use, just download and add the file to your project
